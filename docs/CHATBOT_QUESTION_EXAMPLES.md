# LocalHub AI 연결형 예시 질문

아래 질문은 챗봇 답변의 관련 링크를 통해 지도 또는 축제 캘린더로 이동하도록 지원합니다.

여행 성향 검사를 완료한 사용자는 저장된 `HEALING`, `EXPLORER`, `CULTURE`, `FOODIE` 유형이 모든 장소 질문에 함께 전달됩니다. 질문의 지역·장소 조건을 먼저 만족시킨 뒤 성향 키워드와 태그가 많이 일치하는 장소가 상단에 배치됩니다.
성향은 브라우저에 저장되어 새로고침하거나 다시 접속해도 챗봇 요청에 포함됩니다. 성향이 없을 때는 일반 여행 추천에서 관광지·문화시설·레포츠를 우선하고 대형마트 같은 무관한 쇼핑 장소를 제외합니다.

## 지도 연결 질문

- 금오산 위치를 지도에서 보여줘.
- 동락공원은 지도에서 어디에 있어?
- 구미역 주변 장소를 지도에서 찾아줘.
- 구미 관광지 몇 곳의 위치를 알려줘.
- 경주에 있는 여행지를 지도에서 보여줘.
- 산책하기 좋은 장소 위치를 지도에서 확인하고 싶어.
- 아이와 갈 만한 장소를 지도에서 찾아줘.
- 구미 맛집 위치를 지도에서 보고 싶어.
- 금오산 둘레길의 좌표와 위치를 알려줘.
- 금오천 주변 명소를 지도에서 열어줘.

지도 질문의 장소 링크는 `/map?selected={contentId}`로 이동합니다. 지도는 해당 장소 좌표로 확대하고 상세 정보 창을 엽니다.

## 축제 캘린더 연결 질문

- 이번 달 축제 일정을 캘린더로 보여줘.
- 다음 달 경북 축제를 달력에서 확인하고 싶어.
- 2026년 8월 축제 캘린더를 열어줘.
- 7월에 열리는 구미 축제를 달력으로 보여줘.
- 구미 라면축제 날짜와 캘린더를 알려줘.
- 지금 날짜에서 가장 가까운 축제를 캘린더에서 열어줘.
- 이번 달 행사는 어떤 것이 있는지 일정표로 알려줘.
- 다음 달 이벤트 일정을 달력으로 확인하고 싶어.
- 경주 축제 일정을 캘린더로 보여줘.
- 올해 10월에 진행하는 축제를 달력에서 보고 싶어.

연·월 캘린더 링크는 `/festivals?year={year}&month={month}`로 이동합니다. 특정 축제 링크는 `/festivals?selected={contentId}`로 이동해 해당 월과 축제 상세 정보를 함께 표시합니다.

## 해석 기준

- `이번 달`, `이달`: 현재 날짜의 연·월
- `다음 달`: 현재 날짜의 다음 연·월이며 12월 다음은 다음 해 1월
- `7월`처럼 연도가 없으면 현재 연도
- `지도`, `위치`, `좌표`: 지도 이동 의도
- `캘린더`, `달력`, `일정표`: 축제 캘린더 이동 의도
- 검색 데이터에 없는 장소·축제는 추측하지 않고 정보가 부족하다고 안내
